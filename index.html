RESEARCH EXPERIENCE

  • Regulating Explainability in Machine Learning Applications

    I was selected to be an undergraduate researcher at Carnegie Mellon University in the summer of 2023. I worked with a team of software engineering researchers and sociologists to examine what policies aimed at explainability to regulate machine learning applications should look like. My role for this project was building machine learning models using the library sci-kit learn in Python and then writing descriptions of the models for a non-technical audience. One of our findings was that it is easier to provide explanations when a purpose is clearly stated. The National Science Foundation funded our work.

  • Genetic Algorithm for COVID Model Outcomes

    I programmed a genetic algorithm in Python with two others that transforms machine learning input into a new form. This reesearch was under Professor Adam Eck.

  • Analyzing Short-Term Sea Surface Salinity Varability in relation to Climate

    As part of Oberlin's January Winter Term 2023, I worked with an oceanography researcher to analyze short-term sea surface salinity data and examine how it might correlate with other factors. You can read more about this project here.
