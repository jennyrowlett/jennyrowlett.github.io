RESEARCH EXPERIENCE

  • Regulating Explainability in Machine Learning Applications

    I was selected as an undergraduate researcher at Carnegie Mellon University in summer 2023. I worked on a team of software engineer researchers and sociologists to examine what policies aimed at explainability to regulate machine learning applications should look like. My role for this project was building machine learning models using the library sci-kit learn in Python and then writing descriptions of the models for a non-technical audience. One of our findings was that it is easier to provide explanations when a purpose is clearly stated.

  • Analyzing Short-Term Sea Surface Salinity Varability in relation to Climate

    As part of Oberlin's January Winter Term 2023, I worked with an oceanography researcher to analyze short-term sea surface salinity data and examine how it might correlate with other factors. You can read more about this project here.
